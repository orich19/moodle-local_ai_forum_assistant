Plugin: AI Forum Assistant
Tipo: local
Autor: Richard Odon (Sendatec)
Versión: 0.1 Alpha
Descripción: Esqueleto base para un plugin que automatiza respuestas en foros de Moodle usando IA.

Instrucciones:
1. Copiar la carpeta "ai_forum_assistant" en local/
2. Visitar la página de administración para completar la instalación.
3. Configurar clave API en Administración del sitio > Plugins locales > AI Forum Assistant.
4. Probar la activación en foros (el evento está listo para conectar con IA).
